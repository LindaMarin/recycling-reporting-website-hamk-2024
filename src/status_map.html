Instructions for Navigating the Status Map:

  1. Use the filter buttons at the top to display different sets of issues:
    • All Issues: View all reported issues on the map.
    • Open Issues: Filter to see only currently unresolved issues.
    • In-Progress Issues: Display issues that are being actively addressed.
    • Resolved Issues: Check issues that have been marked as resolved.
  2. To find a specific issue, use the search bar by entering the issue ID. The map will automatically zoom to the issue location.
  3. Click on any issue marker on the map to view more details about the issue.
  4. If necessary, you can mark an issue as resolved or reopen it from the details popup.
  5. Zoom in and out on the map to explore different areas and view reported issues.
